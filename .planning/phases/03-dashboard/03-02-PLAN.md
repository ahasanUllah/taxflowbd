  ```typescript
  interface RecentActivityProps {
    latestReturn: { status: "DRAFT" | "COMPLETE" | "SUBMITTED"; updatedAt: Date } | null
  }
  ```
- Status map (defined outside component): `const statusMap = { DRAFT: "খসড়া", COMPLETE: "সম্পন্ন", SUBMITTED: "দাখিলকৃত" } as const`
- Outer card: `<div className="bg-white rounded-2xl border border-gray-100 p-6 shadow-sm">`
- Section heading: `<h2 className="text-base font-semibold text-foreground mb-4">সাম্প্রতিক কার্যক্রম</h2>`
- **When `latestReturn` is null (empty state):**
  - `<p className="text-sm text-gray-500 mb-4">এখনো কোনো রিটার্ন দাখিল করা হয়নি।</p>`
  - CTA button: `<Link href="/dashboard/return/interview" className="inline-flex items-center border border-brand-primary text-brand-primary rounded-xl px-5 py-2.5 text-sm hover:bg-brand-surface transition">এখনই শুরু করুন</Link>`
- **When `latestReturn` exists:**
  - Status row: `<div className="flex items-center justify-between">` with `<span>অবস্থা:</span>` and `<span className="font-medium text-brand-primary">{statusMap[latestReturn.status]}</span>`
  - Date row: `<div className="flex items-center justify-between mt-2 text-sm text-gray-500">` with `<span>সর্বশেষ আপডেট:</span>` and `<span>{new Date(latestReturn.updatedAt).toLocaleDateString("bn-BD")}</span>`
- Default export: `RecentActivity`
</action>
<acceptance_criteria>
- No `"use client"` directive
- Props: `latestReturn` is `{ status, updatedAt } | null`
- When `latestReturn` is null: renders "এখনই শুরু করুন" as a `<Link>` with href `/dashboard/return/interview`
- When `latestReturn` exists: renders mapped Bangla status (DRAFT→খসড়া, COMPLETE→সম্পন্ন, SUBMITTED→দাখিলকৃত)
